# Lab 16: GRC104 Module

**Course:** GRC104 - Compliance & Legal Requirements  
**Week:** 16  
**Duration:** 5 days  
**Level:** Intermediate  
**Prerequisites:** GRC101-103 completion

## Lab Overview

This lab focuses on compliance frameworks and regulatory requirements. Participants will perform compliance assessments, create audit plans, and develop remediation strategies.

## Learning Objectives

Upon completion of this lab, you will be able to:

- Understand major compliance frameworks
- Perform compliance gap assessments
- Create audit plans and checklists
- Document compliance findings
- Develop remediation roadmaps

## Required Resources

- Compliance audit data (datasets/compliance_audit_data.xlsx)
- Audit checklist template (templates/audit_checklist_template.xlsx)
- Compliance frameworks (resources/frameworks/)
- Lab report template

## Lab Tasks

### Task 1: Framework Mapping and Analysis

Map organizational controls to compliance frameworks:
- NIST CSF mapping
- ISO 27001 alignment
- CIS Controls assessment
- PCI DSS requirements
- Gap identification

### Task 2: Compliance Assessment

Conduct comprehensive compliance assessment:
- Control evaluation
- Evidence collection
- Gap analysis
- Risk assessment
- Remediation planning

### Task 3: Audit Planning and Execution

Develop and execute audit procedures:
- Audit plan creation
- Checklist development
- Evidence gathering
- Finding documentation
- Remediation tracking

## Assessment Criteria

Your lab will be assessed on:

- Framework understanding (20%)
- Assessment completeness (25%)
- Gap analysis accuracy (20%)
- Audit plan quality (20%)
- Documentation quality (15%)

## Submission Requirements

Submit the following by Friday 11:59 PM:

1. Lab Report (PDF or Markdown)
2. Framework mapping documentation
3. Compliance assessment results
4. Audit plan and checklists
5. Findings and remediation plan
