## [1.0.0] - 2025-12-09

### Added
- Initial repository structure with 47 weekly labs
- Phase 1: Compliance & Governance Foundations (18 weeks)
  - GRC101: Introduction to GRC & Technical Landscape (4 labs)
  - GRC102: Information Security Governance (5 labs)
  - GRC103: Risk Assessment & Management (5 labs)
  - GRC104: Compliance & Legal Requirements (4 labs)
- Phase 2: Advanced GRC & Technical Integration (17 weeks)
  - GRC201: Incident Management & BCP (5 labs)
  - GRC202: Security Awareness Engineering (3 labs)
  - GRC203: Cloud GRC & Automation (5 labs)
  - GRC204: Security Operations for GRC Engineers (4 labs)
- Phase 3: GRC Implementation & Automation (12 weeks)
  - GRC301: DevSecOps & IaC (4 labs)
  - GRC302: GRC Tooling & Platform Engineering (4 labs)
  - GRC399: Capstone Project (4 labs)
- Comprehensive datasets
  - risk_assessment_data.xlsx
  - compliance_audit_data.xlsx
  - organizational_data.xlsx
  - system_logs.txt
- Framework documentation
  - NIST Cybersecurity Framework summary
  - ISO 27001:2022 controls
  - CIS Critical Security Controls
  - PCI DSS requirements
- Automation scripts
  - linux_compliance_check.sh
  - password_policy_checker.sh
- Templates directory with reusable templates
- Resources directory with tools guide
- README.md with programme overview
- GETTING_STARTED.md with setup instructions
- SCHEDULE.md with 52-week timeline
- Dataset documentation

### Documentation
- Comprehensive README with programme structure

- Template usage instructions
