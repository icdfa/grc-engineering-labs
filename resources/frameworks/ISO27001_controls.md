# ISO 27001:2022 Controls Summary

## Overview

ISO 27001:2022 is an international standard for information security management systems (ISMS).

## Control Structure

### Four Control Sections

1. **Organizational Controls (A.5)** - 11 controls
2. **People Controls (A.6)** - 8 controls
3. **Physical Controls (A.7)** - 14 controls
4. **Technological Controls (A.8)** - 34 controls

Total: 93 controls across 4 categories

## Key Strengths

- Comprehensive coverage
- International recognition
- Certification available
- Prescriptive guidance
- Widely adopted

## Implementation Approach

1. Establish ISMS policy
2. Define scope
3. Assess risks
4. Select controls
5. Implement controls
6. Monitor and measure
7. Maintain certification
